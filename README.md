# primeiro-projeto-em-grupo

Patas & Lares

TEMA
Animais perdidos e para adoção

PROBLEMA
O abandono e a perda de animais é um problema muito comum nas cidades. Segundo a Organização Mundial da Saúde (OMS), o Brasil possui cerca de 30 milhões de animais abandonados, entre cães e gatos. Muitos desses animais vivem nas ruas, passam fome, sofrem doenças e correm risco de acidentes. Além disso, muitas pessoas encontram animais perdidos, mas não sabem como encontrar os donos ou como colocar o animal para adoção. Por isso, muitas vezes esses animais não conseguem voltar para casa ou encontrar um novo lar. Diante dessa situação, pensamos em criar uma solução simples utilizando um sistema web para ajudar a divulgar animais perdidos e facilitar a adoção.

SOLUÇÃO
Para ajudar a resolver esse problema, criamos o site Patas & Lares. O site funciona como um sistema de cadastro de animais perdidos e também como uma plataforma de adoção. Nele, o usuário pode cadastrar um animal perdido informando nome, tipo, bairro, descrição, telefone e foto. Outras pessoas podem visualizar a lista de animais perdidos e ajudar a encontrá-los. O site também possui uma área para adoção, onde animais que foram encontrados ou estão sem lar podem ser cadastrados para encontrar uma nova família.

OBJETIVO
O principal objetivo do projeto é ajudar animais perdidos a retornarem para seus lares. Além disso, o projeto também tem como objetivo incentivar a adoção responsável, conscientizar as pessoas sobre a importância de cuidar bem dos animais e utilizar a tecnologia como uma ferramenta para ajudar a comunidade. 

FUNCIONALIDADE DO SITE
O site Patas & Lares possui diversas funcionalidades. Entre elas estão: a página inicial com informações sobre o projeto, a página de cadastro de animais perdidos, a lista de animais perdidos com foto e descrição, a página de cadastro de animais para adoção, a lista de animais disponíveis para adoção e um formulário de contato para pessoas interessadas em adotar um animal. O site também possui imagens dos animais e informações de contato para facilitar a comunicação entre as pessoas.

TECNOLOGIAS UTILIZADAS
Para o desenvolvimento do site, utilizamos HTML para criar a estrutura das páginas, como títulos, textos, imagens e formulários. Utilizamos PHP para processar os dados dos formulários, como as informações cadastradas dos animais e os dados das pessoas interessadas em adoção. Também utilizamos CSS, colocamos imagens para mostrar os animais e facilitar a identificação, estilizar o site, definindo cores, layouts e melhorando a aparência visual.
